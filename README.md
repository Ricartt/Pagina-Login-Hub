# Pagina-Login-Hub 
Este projeto consiste em uma página de login elegante e responsiva, projetada para permitir que usuários se conectem a uma plataforma ou sistema. O design utiliza o framework Tailwind CSS e implementa um efeito de vidro fosco para os campos de entrada, além de um layout limpo e moderno.

Descrição

A página de login apresenta uma interface de usuário moderna e minimalista, composta por duas seções principais:

Seção de Login: Contém o formulário com campos para inserir o email e senha, além de um botão de envio com um gradiente de cores vibrantes. O formulário também inclui um link para recuperação de senha.

Seção de Imagem: Uma imagem de fundo é exibida na parte direita da tela, com efeito de zoom e sobreposição escura. A imagem é responsiva e adapta-se bem em diferentes tamanhos de tela.

Funcionalidades:

Validação de Formulário: O formulário possui uma validação simples em JavaScript, que garante que os campos de email e senha sejam preenchidos antes de submeter o formulário.

Botão de Envio: O botão de login apresenta um efeito de gradiente e se destaca ao passar o mouse sobre ele.

Responsividade: O layout foi criado para ser completamente responsivo, utilizando Tailwind CSS, e se adapta a diferentes tamanhos de tela (desktop, tablet e dispositivos móveis).

Efeito Frosted Glass: Os campos de entrada possuem um efeito de vidro fosco, com bordas arredondadas e sombreamento para criar uma sensação visual sofisticada.

Tecnologias Utilizadas:

HTML5: Estruturação do conteúdo da página, incluindo a definição de formulários e campos de entrada.

CSS3 (Tailwind CSS): Framework de estilização utilizado para criar o layout, com ênfase em flexbox e utilitários para responsividade.

JavaScript: Implementação de validação para garantir que os campos sejam preenchidos corretamente antes de enviar o formulário.

Efeito de Imagem: A imagem é manipulada com transições para criar um efeito de zoom suave ao passar o mouse sobre ela.

Como Funciona:

O usuário insere seu email e senha nos campos correspondentes.

Se qualquer campo estiver vazio, o JavaScript irá exibir um alerta pedindo para o usuário preencher o campo faltante.

Ao preencher corretamente os campos, o formulário é enviado e um alerta de login bem-sucedido é exibido.

O usuário é redirecionado para a página bemvindo.html após um login bem-sucedido.
